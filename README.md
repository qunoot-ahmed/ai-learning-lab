# AI Learning Lab

A hands-on repository documenting my AI engineering journey using Python and Google's Gemini API.

## Tech Stack

- Python
- Gemini API
- google-genai
- python-dotenv
- Virtual Environment (.venv)

## Completed

### 01 First AI App
- Connected Python to Gemini API
- Stored API key using environment variables
- Sent a prompt to Gemini
- Displayed AI response in terminal

### 02 Chatbot
- Built an interactive CLI chatbot
- Added continuous chat loop
- Added input validation
- Added help and exit commands
- Used command handling with a dictionary

### 03 Conversation Memory
- Added conversation history using a list
- Stored user prompts and model responses with roles
- Sent full conversation history to Gemini
- Enabled the chatbot to remember previous messages

## Security

API keys are stored in `.env` and excluded from Git using `.gitignore`.
